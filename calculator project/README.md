
# Calculator Project

A simple and responsive calculator built using **HTML** and **CSS**. This project demonstrates the basics of web design, layout creation, and styling.

## 🚀 Features

- Clean and modern user interface
- Responsive design
- Basic calculator layout
- Built with pure HTML and CSS
- Beginner-friendly project

## 🛠️ Technologies Used

- HTML5
- CSS3

